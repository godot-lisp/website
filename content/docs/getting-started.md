# Getting Started

Welcome to GOLI! This guide will help you get up and running with Lisp programming in Godot.

## Prerequisites

Before you begin, ensure you have:
- Godot 4.x installed
- Basic familiarity with Godot's editor
- Understanding of Lisp concepts (helpful but not required)

## Installation

1. Download the GOLI extension
2. Install the GDExtension in your Godot project
3. Configure project settings

## Your First Lisp Script

1. Create a new scene
2. Add a Node with a script
3. Write your first Lisp function
4. Test the script
